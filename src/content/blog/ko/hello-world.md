## 첫 글

이 블로그는 [Astro](https://astro.build)로 만들었습니다. 미니멀하고 텍스트 중심의
공간을 목표로 합니다.

### 앞으로 다룰 것

- 개발하며 배운 것들
- 만든 프로젝트 기록
- 가끔의 잡담

```ts
// 코드 블록도 이렇게 예쁘게 표시됩니다.
function hello(name: string): string {
  return `안녕하세요, ${name}!`;
}
```

> 꾸준히 기록하는 것이 목표입니다.
